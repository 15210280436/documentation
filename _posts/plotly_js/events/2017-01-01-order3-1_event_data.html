--- name: Event Data plot_url: language: plotly_js suite: events order: 3.1 sitemap: false arrangement: horizontal markdown_content: | Many Plotly events emit event data when the event is triggered. Event data is information about the data point related to the event (i.e. the point clicked).
The following events emit event data: * `plotly_click` * `plotly_hover` * `plotly_unhover` * `plotly_selecting` * `plotly_selected`
Event data differs depending on the type of plot the user is interacting with. The event data structure for Cartesian (2D) plots, 3D plots, and maps can be found below, along with examples of how to use event data. ---
