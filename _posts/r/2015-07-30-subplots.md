# Subplots


```r
# Basic subplot
library(plotly)
subplot(
  plot_ly(economics, x = date, y = uempmed, filename="r-docs/basic-subplot"),
  plot_ly(economics, x = date, y = unemploy),
  margin = 0.05
) %>% layout(showlegend = FALSE)
```
