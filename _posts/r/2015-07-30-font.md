# Custom Font


```r
library(plotly)
f <- list(
  family = "Courier New, monospace",
  size = 18,
  color = "#7f7f7f"
)
plot_ly(x = 0:8, y = 0:8) %>%
  layout(title = "Global Font", font = f)
```
