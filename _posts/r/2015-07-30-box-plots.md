### Basic Boxplot


```r
library(plotly)
plot_ly(y = ~rnorm(50), type = "box") %>%
  add_trace(y = ~rnorm(50, 1))
```

### Adding Jittered Points


```r
plot_ly(y = ~rnorm(50), type = "box", boxpoints = "all", jitter = 0.3,
        pointpos = -1.8)
```

### Grouped Box Plots


```r

  layout(boxmode = "group")
```
