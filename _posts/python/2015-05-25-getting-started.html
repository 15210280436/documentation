Installation

To install Plotly's Python package, use the package manager pip inside your terminal. 

$ pip install plotly

or 

$ sudo pip install plotly

If you don't have pip installed on your machine, click here for pip's installation instructions.

Plotly's Python package is updated frequently! To upgrade, run: 

$ pip install plotly --upgrade

Initialization

In the terminal, copy and paste the following to install the Plotly library and set your user credentials! 

$ pip install plotly
$ python -c "import plotly; plotly.tools.set_credentials_file(username='Nicole_G', api_key='zd4g6c702b', stream_ids=['llyogkx30k', '4vjnpg0p9y'])"

Start plotting!

Fire up Python! 

$ python

Copy and paste the following to create your first Plotly graph using the Plotly Python library: 

import plotly.plotly as py
from plotly.graph_objs import *

trace0 = Scatter(
    x=[1, 2, 3, 4],
    y=[10, 15, 13, 17]
)
trace1 = Scatter(
    x=[1, 2, 3, 4],
    y=[16, 5, 11, 9]
)
data = Data([trace0, trace1])

unique_url = py.plot(data, filename = 'basic-line')

You can also create plotly graphs with matplotlib syntax. Learn more in our matplotlib documentation.

You can also create plotly graphs within an IPython notebook. Learn more in our IPython notebook documentation.

Credentials

The initialization step places a special .plotly/.credentials file in your home directory. Your ~/.plotly/.credentials file should look something like this: 

{
    "username": "Nicole_G",
    "stream_ids": ["llyogkx30k", "4vjnpg0p9y"],
    "api_key": "zd4g6c702b"
}

You can change the contents of this file manually or as described in the Initialization section.
