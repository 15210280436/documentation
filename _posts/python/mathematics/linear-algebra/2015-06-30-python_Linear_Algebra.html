A Matrix is a 2D array that stores real or complex numbers. A Real Matrix is one such that all its elements $r$ belong to $\mathbb{R}$. Likewise, a Complex Matrix has entries $c$ in $\mathbb{C}$

Add Two Matrices¶

How to find the sum of two matrices

In [21]:
matrix1 = np.matrix(
    [[1, 4],
     [2, 0]]
)

matrix2 = np.matrix(
    [[-1, 2],
     [1, -2]]
)

matrix1 + matrix2
Out[21]:
matrix([[ 0,  6],
        [ 3, -2]])

Solve Matrix Equation¶

How to find the solution of $AX=B$

In [23]:
A = np.matrix(
    [[1, 4],
     [2, 0]]
)

B = np.matrix(
    [[-1, 2],
     [1, -2]]
)

X = np.linalg.solve(A, B)
X
Out[23]:
matrix([[ 0.5  , -1.   ],
        [-0.375,  0.75 ]])

How to find the determinent of a matrix

det
