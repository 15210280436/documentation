Introduction¶

In streamtube plots, attributes inlcude x, y, and z, which set the coorindates of the vector field, and u, v, and w, which sets the x, y, and z components of the vector field. Additionally, you can use starts to determine the streamtube's starting position. Lastly, maxdisplayed determines the maximum segments displayed in a streamtube.

Starting Position and Segments¶

In [3]:
import plotly.plotly as py
import plotly.graph_objs as go

import pandas as pd

df = pd.read_csv('https://raw.githubusercontent.com/bcdunbar/datasets/master/streamtube-wind.csv').drop(['Unnamed: 0'],axis=1)

data = [{
    'type': 'streamtube',
    'x': df['x'],
    'y': df['y'],
    'z': df['z'],
    'u': df['u'],
    'v': df['v'],
    'w': df['w'],
    'starts': {
        'x': [80] * 16,
        'y': [20,30,40,50] * 4,
        'z': [0,0,0,0,5,5,5,5,10,10,10,10,15,15,15,15]
    },
    'sizeref': 0.3,
    'colorscale': 'Portland',
    'showscale': False,
    'maxdisplayed': 3000
}]

layout = {
    "scene": {
        "aspectratio": {
            "x": 2,
            "y": 1,
            "z": 0.3
        }
    },
    "margin": {
        "t": 20,
        "b": 20,
        "l": 20,
        "r": 20
    }
}

fig = {'data':data,'layout':layout}
py.iplot(fig, filename="streamtube wind", validate=False)
