Plotly Python Open Source Graphing Library for Pandas

Cufflinks binds plotly to pandas dataframes in IPython notebook.
Plotly is now on Version 3. In order to use cufflinks with Plotly V3 ensure that your cufflinks version 0.14.0 or higher. Run the following command in OS X and Linux to upgrade to the correct cufflinks version:

pip install cufflinks==0.14.0
