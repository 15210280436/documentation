# Log Axes in R


```r
library(plotly)
d <- diamonds[sample(nrow(diamonds), 1000), ]

# without log scales
(p <- plot_ly(d, x = carat, y = price, mode = "markers"))
```

```r
# with log scales
layout(p, xaxis = list(type = "log"),
       yaxis = list(type = "log"))
```
