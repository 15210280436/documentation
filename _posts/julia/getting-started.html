Getting started: Plotly for Julia

Installation

Install using Julia's Pkg module. From the Julia REPL, do: 

Pkg.clone("https://github.com/plotly/Plotly.jl")

Initialization

In the Julia console, begin by importing the Plotly Julia library. 

using Plotly

Authentication

Sign in to your Plotly Julia library account.

You'll need to replace "DemoAccount" and "lr1c37zw81" with your Plotly username and API-key. 

Plotly.signin("DemoAccount", "lr1c37zw81")

Julia will return:

PlotlyAccount("DemoAccount", "lr1c37zw81")

If you don't have an account yet you can sign up using Plotly's online GUI.

Don't know your API key?

Display Key

It is also possible to save your user credentials for future Julia sessions. In your Julia terminal, enter: 

Plotly.set_credentials_file({"username"=>"DemoAccount","api_key"=>"lr1c37zw81"})

Start plotting!

Now you can make a simple API call using Plotly.plot(), and create your first graph! 

x = [0,1,2,3,4]
y = [0,3,2,4,5]
Plotly.plot([x y])

Response

Here is the response you'll receive (copy and paste the URL into your browser to see your graph!) 

["url"=>"http://plot.ly/~username/888",
"filename"=>"filename",
"error"=>"",
"message"=>"",
"warning"=>""]
